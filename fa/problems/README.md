# مسائل نظام سلامت

این بخش محل نگهداری مسائل ساختاری نظام سلامت است.

هدف ثبت شکایت نیست؛ هدف تبدیل تجربه‌ها و مشکلات پراکنده به مسئله‌های قابل فهم، تحلیل و اصلاح است.

هر مسئله باید بررسی کند:

- چه اتفاقی رخ می‌دهد؟
- چه افرادی تحت تأثیر قرار می‌گیرند؟
- ریشه‌های مشکل چیست؟
- پیامدهای کوتاه‌مدت و بلندمدت چیست؟
- چه راه‌حل‌هایی قابل تصور است؟

## حوزه‌های اولیه بررسی

### رابطه بیمار، پزشک و بیمه

بازطراحی ارتباط میان:
- هزینه درمان
- کیفیت خدمت
- اعتماد
- پرداخت عادلانه
- نقش بیمه


### آموزش پزشکی و عدالت فرصت

موضوعاتی مانند:
- کیفیت آموزش
- مسیر ورود
- سهمیه‌ها
- ارزیابی صلاحیت حرفه‌ای


### سلامت روان کادر درمان

موضوعاتی مانند:
- فرسودگی شغلی
- فشار روانی
- کاهش انگیزه
- مهاجرت


### زیرساخت دیجیتال سلامت

موضوعاتی مانند:
- پرونده سلامت الکترونیک
- نسخه‌نویسی دیجیتال
- هوش مصنوعی پزشکی
- تحلیل داده


هدف نهایی:

تبدیل درد فردی → فهم جمعی → اصلاح سیستم
