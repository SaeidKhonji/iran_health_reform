# Healthcare Problems

This section contains structured analysis of healthcare system problems.

The purpose is not to collect complaints.

The purpose is to transform isolated experiences into:

- understandable problems
- root cause analysis
- possible solutions
- institutional learning

---

## How problems are analyzed

Each problem should explore:

1. What is happening?
2. Who is affected?
3. Why does it happen?
4. What are the short-term and long-term consequences?
5. What solutions are possible?
