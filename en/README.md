# Iran Health Reform Knowledge Base

This section contains the English knowledge base of the project.

The goal is to create a collaborative, evolving understanding of healthcare system problems and possible reforms.

---

## Philosophy

Healthcare problems are rarely caused by a single person or group.

They usually emerge from interactions between:

- individuals
- institutions
- incentives
- culture
- technology
- economic structures

A systems approach tries to understand these relationships before proposing solutions.
