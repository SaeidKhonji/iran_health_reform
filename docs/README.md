# Knowledge Base

This folder contains the evolving knowledge structure of the Iran Health Reform project.

The goal is to transform:

Experience → Problem → Analysis → Solution → Implementation → Evaluation

## Structure

### 01_experiences

Real experiences from:
- patients
- physicians
- nurses
- healthcare workers
- managers

Purpose:
Turning individual stories into collective memory.

---

### 02_problems

Structured definitions of healthcare problems.

Each problem should include:

- description
- affected groups
- causes
- consequences
- available data

---

### 03_analysis

Systems-level analysis:

- biological factors
- psychological factors
- social factors
- economic factors
- institutional factors
- cultural factors

---

### 04_solutions

Possible reforms and interventions.

Solutions should be:

- realistic
- measurable
- testable
- open to criticism

---

### 05_projects

Practical pilots and implementation plans.

Ideas become action here.
